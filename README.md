This is the repository for my personal resume, built with pure HTML and CSS. The project is designed with a focus on professional aesthetics, readability, and pixel-perfect PDF export.

## 🌟 Features

- **Responsive Design:** The resume looks great on all devices (mobile, tablet, desktop).
- **Print Optimization:** Uses `@media print` rules to ensure the resume exports as a perfectly formatted A4 PDF document.
- **Modern Typography:** Features the *Inter* font family from Google Fonts.
- **Interactivity:** Quick "Save as PDF" button for easy document export.
- **SEO Ready:** Optimized Meta tags and Open Graph support for social sharing.

## 🛠 Tech Stack

- **Structure:** HTML5 (Semantic elements)
- **Styling:** Vanilla CSS3 (Custom properties/variables, Flexbox, Grid)
- **Icons:** FontAwesome 6
- **Fonts:** Google Fonts (Inter)

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/ods-fsd/my-cv.git
   ```
2. Navigate to the project directory:
   ```bash
   cd my-cv
   ```
3. Open `index.html` in your favorite web browser.

## 📄 Exporting to PDF

To get a hard copy or a file for applications:
1. Open the page in your browser.
2. Click the **"Save as PDF"** button in the top-right corner.
3. In the print dialog, select **"Save as PDF"** as the destination.
4. Ensure the paper size is set to **A4** and (optionally) disable "Headers and footers" for a cleaner look.
